# MKS-LCD-12864-v3-with-Ender3-v2
How to make the mks LCD 12864 v3 display work with the ender3v2 stock board
![scrren and printer](https://github.com/user-attachments/assets/2388d4bf-0a2c-4baf-8190-40545fb32ae5)

When my original display died i hopped on aliexpress and bought myself 20ish hours of work to cheap out on 15 euros extra, but i guess it looks pretty...

This guide includes:
- firmware config
- pinout and how to prepare the custom cabling
- mistakes and solutions i found out

## Related Hardware

- Ptinter: Ender 3v2
- Main boards: 4.2.2 
- Display: MKS-LCD-12864-v3 
- Firmware: Marlin 2.x
- Compiling done in: VSCode + PlatformIO + autobuildmarlin
